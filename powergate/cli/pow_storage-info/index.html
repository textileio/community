    Storage Info

    pow storage-info¶

    Provides commands to get and query cid storage info.

    Synopsis¶

    Provides commands to get and query cid storage info.

    Options¶

      -h, --help   help for storage-info

    Options inherited from parent commands¶

          --serverAddress string   address of the powergate service api (default "127.0.0.1:5002")
  -t, --token string           user auth token

    SEE ALSO¶

    • pow - A client for storage and retreival of powergate data
    • pow storage-info get - Returns the information about a stored cid.
    • pow storage-info list - Returns a list of information about all stored cids, filtered by cids if provided.
