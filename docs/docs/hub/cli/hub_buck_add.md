# hub buck add

Add adds a UnixFs DAG locally at path, merging with existing content.

```
hub buck add [cid] [path] [flags]
```

### Options

```
  -h, --help   help for add
  -y, --yes    Skips confirmations prompts to always overwrite files and merge folders
```
